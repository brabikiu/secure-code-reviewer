# Secure Code Reviewer

Extensión de VS Code para analizar calidad y seguridad en código JavaScript y TypeScript.

## Funcionalidades iniciales

- Detecta uso de `eval`
- Detecta asignaciones a `innerHTML`
- Detecta posibles secretos hardcodeados
- Detecta `console.log`
- Muestra diagnósticos en Problems
- Ofrece Quick Fixes básicos

## Comandos

- Secure Code Reviewer: Analizar archivo actual
- Secure Code Reviewer: Analizar workspace
